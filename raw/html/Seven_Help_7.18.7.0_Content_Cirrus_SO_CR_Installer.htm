Using the Cirrus Agent Installer

 

Summary

 

In order to help Cirrus detect and connect to your peripheral hardware, like receipt printers and cash drawers, you need to download and install the Cirrus Agent on any workstation that will be used to work in Cash Receipts. This small application will run in the background when you are working in Cirrus CR and ensure a consistent connection between Cirrus and your peripheral hardware.

 

NOTE: If you do not have access to the Install Cirrus Agent page in Cirrus, you can use the direct link below to download, install, and run the agent.

Cirrus Agent direct download link

 

Step by Step

 

  • Open the Install Cirrus Agent page (Support> Install Cirrus Agent).

  • Click the Download button to download the Cirrus Agent Installer.msi file.

  • Once the file has successfully downloaded, browse to the download location and double-click the file to open the Cirrus Agent Setup Wizard.

  • Click the Install button and follow the prompts to install the Cirrus Agent.

  • Click the Finish button when the installation is complete.

  • Once installed, the Cirrus Agent will automatically start running every time the workstation is in use.
